# MiniSearch

## Overview

MiniSearch is a lightweight, in-memory full-text search engine for JavaScript.

## Key Features

- Zero dependencies
- Supports fuzzy matching and prefix search
- BM25-based scoring
- Works with [[vault-engine]] as the search backend
