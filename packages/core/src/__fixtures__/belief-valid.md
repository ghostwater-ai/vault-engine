# Structured Knowledge Improves Agent Performance

## Summary

We believe that AI agents perform better when they have access to structured, typed knowledge bases rather than flat document stores.

## Key Claims

- Type-aware retrieval reduces hallucinations
- Epistemic metadata enables confidence-weighted responses
- See [[type-system-design]] for our approach

## Supporting Evidence

Our experiments with [[vault-engine-prototype]] show 23% improvement in accuracy.
